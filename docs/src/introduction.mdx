# React-ellipsis-component

A high performance and customized ellipsis component for react. Support to custom ellipsis char, ellipsis html, end char filter, reflow callback, ...etc.

Github Repo: [react-ellipsis](https://github.com/chenquincy/react-ellipsis) (Looking forward your star 😁)

![npm](https://img.shields.io/npm/v/react-ellipsis-component.svg) ![Download](https://img.shields.io/npm/dt/react-ellipsis-component.svg) ![lang](https://img.shields.io/badge/language-typescript-yellow.svg)

> Reminder: Demo container can be resize by dragging in docs.
