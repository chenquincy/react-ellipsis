# React-ellipsis-component

高性能、可自定义的 react 文本缩略组件。支持自定义缩略符、html 自定义缩略符、尾文本过滤、缩略回调等功能。

github 仓库: [react-ellipsis](https://github.com/chenquincy/react-ellipsis) (求 star 😁)

![npm](https://img.shields.io/npm/v/react-ellipsis-component.svg) ![下载数量](https://img.shields.io/npm/dt/react-ellipsis-component.svg) ![语言](https://img.shields.io/badge/language-typescript-yellow.svg)

> 提示：在文档中，为了方便展示，可用鼠标拖动容器右下角调整宽度，实际组件不带调整宽度功能。
